Exercícios de laços e Funções

Selecione uma atividade e pratique seus conceitos:

  1. For - Contar de 1 a 10

    Use um loop 'for' para imprimir no console os números de 1 a 10.

  2. While - Soma de 1 a 50

    Usando um loop 'while', some todos os números de 1 até 50 e exiba o resultado com console.log().

  3. For - Contagem regressiva

    Peça ao usuário (via prompt()) um número. Use um 'for' para fazer uma contagem regressiva até 0, mostrando cada número no console.

  4. Do...While - Repetir até digitar "ok"

    Use um loop 'do...while' que fique pedindo ao usuário digitar qualquer coisa até que ele escreva "ok".

  5. Função + For - Tabuada de um número

    Crie uma função que recebe um número informado pelo usuário. A função deve imprimir a tabuada de 1 a 10 desse número usando um 'for'.

  6. While - Contar números pares até 100

    Use um loop 'while' para percorrer os números de 1 a 100. Conte quantos números são pares e exiba o total com console.log().

  7. For - Mostrar cada caractere de uma palavra

    Peça uma palavra ao usuário com prompt(). Use um 'for' para exibir cada caractere da palavra separadamente no console

  8. Função - Calcular o quadrado de um número

    Crie uma função chamada quadrado() que recebe um número informado pelo usuário e retorna ele multiplicado por ele mesmo. Exiba o valor retornado com console.log().

  9. For - Repetir uma frase N vezes

    Peça ao usuário uma frase e um número (N). Use um 'for' para imprimir a frase no console N vezes

  10. While - Sistema de senha com 3 tentativas

    Peça ao usuário uma senha (ex: "1234") usando prompt(). Permita até 3 tentativas usando um loop 'while'. Se acertar, exiba "Senha correta!". Se errar 3 vezes, exiba "Acesso bloqueado".
